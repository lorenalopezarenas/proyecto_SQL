# SQL_proyecto: Alquiler de Películas


## Resumen
Proyecto de SQL con una base de datos de alquiler de películas. Diferentes ejercicios propuestos para
    - hacer consultas con una sola tabla de la BBDD
    - crear relaciones entre tablas
    - realizar subconsultas
    - crear vistas
    - utilizar estructuras de datos temporales
    - buenas prácticas


## Esquema
- Tablas principales: customer, film, category, film_category, inventory, rental, actor, film_actor
